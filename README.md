# Seguimiento Comercial AutoTrato + Google Sheets (Administrable)

Esta versión mueve la administración operativa a Google Sheets.

## Catálogos administrables
- `Vendedores`
- `Indicadores`

## Registro automático
- `Mediciones`
- `Detalle`

## Flujo
1. Editas vendedores e indicadores en Sheets.
2. La web recarga los catálogos.
3. El coordinador carga mediciones.
4. El dashboard resume la información mensual.
